# TextPlus
We are building a simple GUI based text editor with LSP support and syntax highlighting.

The text editor is called TextPlus.

We will be using C# and Avalonia UI.

I have initialized the basic C# project. There is nothing else added.

The app will have black and gray neomorphic styling.

Document your progress in a `.txt` file, and plece it in the `/history` directory.
Make sure all the files are named like this `fileName_MM-DD-YYYY.txt`.
You can catch up with the current progress, by reading the latest entry in the `/history` directory.

You will find feature goals in the `features.md` file.
Any features there that you have worked on, mark the progress on that feature in that file.
